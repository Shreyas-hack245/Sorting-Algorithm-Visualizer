# Sorting Algorithm Visualizer

A simple Python project that visualizes sorting algorithms using Pygame.

## Features

* Bubble Sort
* Selection Sort
* Insertion Sort
* Animated sorting visualization
* Comparison and swap counting

## Technologies Used

* Python
* Pygame

## How to Run

Install Pygame:

```bash
python -m pip install pygame
```

Run the project:

```bash
python main.py
```

## Files

* `main.py` - Visualization and UI
* `algorithms.py` - Sorting algorithms
